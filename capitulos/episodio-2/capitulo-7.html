Episodio 2, Capítulo 7

Instalación de Visual Studio Code

⏱️ Tiempo de lectura: 4 minutos
🧩 Actividades: 1 (Descarga)
📚 Nivel: Principiante

📝 El Taller del Programador: Nuestro Editor de Código

Ya tenemos Python instalado y nuestro entorno virtual activado y con Django adentro. ¡Estamos avanzando a pasos agigantados! Ahora necesitamos un lugar cómodo y eficiente para escribir todo el código que dará vida a nuestra aplicación Django. Este lugar es un editor de código.

Podrías, en teoría, escribir código en el Bloc de Notas de Windows o en cualquier editor de texto simple. Pero eso sería como intentar construir un mueble fino usando solo un cuchillo de cocina. ¡Se puede, pero no es lo ideal y te complicarías la vida innecesariamente!

Un buen editor de código es esencial para un programador, como un buen taller para un artesano.

🚀 Visual Studio Code: Nuestro Elegido

Para este tutorial, y como recomendación general para tus futuros proyectos, vamos a utilizar Visual Studio Code (VS Code). Es un editor de código desarrollado por Microsoft, ¡pero no te asustes!, es completamente gratuito, de código abierto (open source) y funciona de maravilla en Windows, Mac y Linux.

¿Por qué VS Code?

  • Es muy popular: Una comunidad enorme significa muchos tutoriales, extensiones y ayuda disponible.
  • Es potente y personalizable: Tiene resaltado de sintaxis (colorea tu código para que sea más fácil de leer), autocompletado inteligente, depuración integrada (para encontrar errores) y miles de extensiones para añadirle más funcionalidades.
  • Integración con la terminal: ¡Puedes tener una terminal abierta dentro del mismo editor! Esto es súper cómodo.
  • Ligero y rápido: A pesar de todas sus funciones, es bastante ágil.

📥 Instalando Visual Studio Code: ¡Más Fácil Imposible!

La instalación de VS Code es un paseo por el parque. Sigue estos simples pasos:

  1. Busca en Google: Abre tu navegador web favorito y busca "Visual Studio Code Download" o "Descargar Visual Studio Code".

    Una búsqueda simple te llevará al sitio correcto.

  2. Sitio Oficial: ¡Mucho cuidado aquí! Asegúrate de entrar al sitio web oficial. La dirección web debería ser code.visualstudio.com. Evita descargar de sitios no oficiales para no llevarte sorpresas desagradables (como virus o software no deseado).

    La página oficial de descarga de VS Code. Busca el botón para tu sistema operativo.

  3. Descarga la Versión Correcta: La página detectará automáticamente tu sistema operativo (Windows, Mac o Linux) y te ofrecerá el instalador adecuado. Haz clic en el botón de descarga grande y vistoso.
  4. Instala como Cualquier Otro Programa: Una vez descargado el archivo (será un .exe en Windows, un .dmg en Mac, o un .deb/.rpm en Linux, entre otros), simplemente ejecútalo y sigue las instrucciones. La instalación es muy estándar: aceptar los términos, siguiente, siguiente, instalar... ¡y listo! Generalmente no hay configuraciones raras que hacer.

¡Ya está! En unos pocos minutos tendrás instalado uno de los editores de código más potentes y versátiles del planeta, listo para que empecemos a escribir el código de nuestra aplicación Django.

Te recomiendo ver el video de este capítulo donde te guío visualmente por el proceso de descarga e instalación.

Pequeño Resumen del Capítulo 7 (Episodio 2)

Para escribir nuestro código Django, usaremos Visual Studio Code (VS Code), un editor potente y gratuito. Para instalarlo, simplemente búscalo en Google ("Visual Studio Code Download"), asegúrate de entrar al sitio oficial (code.visualstudio.com), descarga la versión para tu sistema operativo y sigue los pasos del instalador. ¡Es muy sencillo!

Con Python, Django y VS Code instalados y listos, ya tenemos todo nuestro "kit de desarrollo" preparado. En el próximo capítulo, daremos el emocionante paso de crear nuestro primer proyecto Django. ¡La aventura continúa!
